# Cheat Sheet

## Reset Inputs

```
sudo ~/RetroPie-Setup/retropie_packages.sh emulationstation init_input
```

## Controls

Keyboard:

* A: A
* B: B
* X: X
* Y: Y
* Start: Enter
* Select: Space
* LS: Left Ctl
* RS: Right Ctl
* LT: Left Alt
* RT: Right Alt
* Hotkey: H

Hotkeys: https://retropie.org.uk/docs/Controller-Configuration/
