Virtual Keyboard:

* Right ctrl / Keyboard:ASR changes port
* RST - Reset
* ZOM - Turbo trigger
* ASR - Aspect Screen Ratio
* JOY - Port
* SVD - Save Disk
* STB - Status Bar
