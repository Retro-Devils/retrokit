## Controls

Instructions:

* https://docs.libretro.com/guides/input-and-controls/#keyboard-controls
* https://retropie.org.uk/docs/Commodore-64-VIC-20-PET/#controls_1

Virtual Keyboard:

* Right ctrl / Keyboard:ASR changes port
* RST - Reset
* ZOM - Turbo trigger
* ASR - Aspect Screen Ratio
* JOY - Port
* SVD - Save Disk
* STB - Status Bar

### 180

Keys:

* 1 - One player
* 2 - Two player
* 3 - Timed round

### Frogger

Keys:

* Shift+F1 - Start

### Pitfall
